Security Policy
Supported Versions

This project provides security updates for the versions listed below. Versions not listed or marked as unsupported may have unpatched vulnerabilities and are not recommended for use.

Version	Supported
5.1.x	:white_check_mark:
5.0.x	:x:
4.0.x	:white_check_mark:
< 4.0	:x:

We recommend upgrading to a supported version to ensure you receive the latest security fixes.

Reporting a Vulnerability

We take security issues seriously. If you discover a vulnerability, please follow these steps:

Contact Method
Email us at beeprotego@gmail.com with the subject line: [Security] Vulnerability Report.
Information to Include
Steps to reproduce the issue
Impact assessment (what could happen if exploited)
Version(s) affected
Response Timeline
Acknowledgement: Within 48 hours
Initial assessment: Within 5 business days
Updates: We will keep you informed until the issue is resolved
Resolution
If verified, we will prioritize a fix and release a patched version
If not valid, we will provide an explanation and close the report
Responsible Disclosure
Do not publicly disclose the issue until a fix is released
Coordinated disclosure helps protect users and maintain trust
